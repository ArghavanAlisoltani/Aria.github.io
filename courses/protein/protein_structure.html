1) Why protein structure matters in real life

biology

Sequence tells you what residues exist. Structure tells you where they sit, which partners they can touch, whether a pocket is open, whether a loop is flexible, and whether a mutation lands in a buried core, a catalytic site, an interface, or a disordered tail.

Hemoglobin

A classic quaternary-structure example: oxygen binding is cooperative because subunits influence one another.

p53

Some mutations destabilize the fold, others mainly disrupt DNA binding. Structure helps separate those mechanisms.

SARS-CoV-2 spike

Small local substitutions can alter receptor binding or antibody recognition without changing the whole fold.

AMR proteins

Porins, enzymes, and targets can change drug access or inhibitor binding through local structural variation.

The strongest structural question is usually mechanistic: which structural change best explains the phenotype?

2) Protein structure hierarchy and amino-acid chemistry

fundamentals

Protein structure is commonly described at four levels: primary (sequence), secondary (α-helices, β-sheets, turns), tertiary (one-chain fold), and quaternary (multi-chain assembly).

LevelMeaningTypical interpretation
PrimaryResidue orderMutation positions, motifs, signal peptides
SecondaryLocal backbone patternHydrogen bonding and Ramachandran preferences
TertiaryWhole-chain foldCore packing, loops, domains, pockets
QuaternaryAssemblyInterfaces, symmetry, allostery, cooperativity
Local backbone geometry is summarized by the torsion angles \( \phi \), \( \psi \), and \( \omega \). Ramachandran-allowed regions correspond to sterically and energetically favorable conformations.
A buried glycine-to-tryptophan substitution is usually more disruptive than an alanine-to-valine change on a solvent-exposed helix.

3) Experimental structure methods and their caveats

experiment

Protein structures come from experiments as well as computation. The main experimental families are X-ray crystallography, cryo-EM, and NMR.

MethodStrengthCommon caveat
X-rayOften very high atomic detailFlexible loops may be missing; crystal packing can bias conformations
Cryo-EMExcellent for large complexesLocal resolution may vary across the map
NMREnsemble view of smaller proteinsOften represented as multiple plausible conformers
The B-factor (temperature factor or Debye–Waller factor) in protein structure represents the displacement or uncertainty of an atom's position around its mean position, indicating local flexibility and mobility. High B-factors (often red/yellow) indicate flexible, disordered regions, while low values (blue) indicate rigid, well-ordered, stable regions. In crystallography, the B-factor is often written as \[ B = \frac{8\pi^2}{3}\langle u^2 \rangle \] where \( \langle u^2 \rangle \) is a mean-square displacement term.
Missing coordinates do not necessarily mean the biology is absent. They often mean that region was too mobile or poorly resolved.

4) Files, databases, and identifiers

data

The most important coordinate formats are PDBx/mmCIF and legacy PDB. mmCIF is the modern flexible standard. You also encounter predicted-model files, ranking JSON files, and confidence outputs such as pLDDT and PAE.

Typical identifiers

PDB ID, chain ID, residue number, insertion code, model number, ligand name, and author numbering versus canonical biological numbering.

Why numbering matters

Many mistakes come from mapping a mutation to the wrong residue index because file numbering and sequence numbering differ.

Annotated Python: download and parse an mmCIF file 
import os
import requests
from Bio.PDB import MMCIFParser

pdb_id = "4HHB"
url = f"https://files.rcsb.org/download/{pdb_id}.cif"
out_file = f"{pdb_id}.cif"

if not os.path.exists(out_file):
    r = requests.get(url, timeout=60)
    r.raise_for_status()
    with open(out_file, "wb") as fh:
        fh.write(r.content)

parser = MMCIFParser(QUIET=True)
structure = parser.get_structure(pdb_id, out_file)

print("Structure ID:", structure.id)
for model in structure:
    print("Model:", model.id)
    for chain in model:
        residues = [res for res in chain if res.id[0] == " "]
        print(f"Chain {chain.id}: {len(residues)} standard residues")

6) Distances, angles, and dihedrals

math + code

Protein structure is fundamentally geometry in 3D. Distances, angles, and torsions are the building blocks behind bond geometry, backbone conformation, residue contacts, and structure comparison.

Distance: \[ d(\mathbf a, \mathbf b) = \sqrt{(a_x-b_x)^2 + (a_y-b_y)^2 + (a_z-b_z)^2} \]
Angle: \[ \theta = \cos^{-1}\left(\frac{\vec{u}\cdot\vec{v}}{\|\vec{u}\|\|\vec{v}\|}\right) \]
Dihedral: the signed angle between two planes defined by four ordered points. This is how backbone \( \phi \), \( \psi \), and side-chain \( \chi \) angles are computed.

Numerical intuition: if points A and B are at \( (0,0,0) \) and \( (1.5,0,0) \), then the A–B distance is 1.5 Å. If A–B–C forms a right angle, then \( \theta = 90^\circ \).

Annotated Python: geometry helper functions 
import numpy as np

def distance(a, b):
    a = np.asarray(a, dtype=float)
    b = np.asarray(b, dtype=float)
    return np.linalg.norm(a - b)

def angle(a, b, c):
    ba = np.asarray(a, float) - np.asarray(b, float)
    bc = np.asarray(c, float) - np.asarray(b, float)
    cosang = np.dot(ba, bc) / (np.linalg.norm(ba) * np.linalg.norm(bc))
    cosang = np.clip(cosang, -1.0, 1.0)
    return np.degrees(np.arccos(cosang))

def dihedral(p0, p1, p2, p3):
    p0, p1, p2, p3 = map(lambda x: np.asarray(x, float), (p0, p1, p2, p3))
    b0 = -(p1 - p0)
    b1 = p2 - p1
    b2 = p3 - p2
    b1 /= np.linalg.norm(b1)
    v = b0 - np.dot(b0, b1) * b1
    w = b2 - np.dot(b2, b1) * b1
    x = np.dot(v, w)
    y = np.dot(np.cross(b1, v), w)
    return np.degrees(np.arctan2(y, x))

A = [0.0, 0.0, 0.0]
B = [1.5, 0.0, 0.0]
C = [1.5, 1.5, 0.0]
D = [2.2, 1.9, 1.1]

print("Distance A-B:", round(distance(A, B), 3), "Å")
print("Angle A-B-C:", round(angle(A, B, C), 3), "degrees")
print("Dihedral A-B-C-D:", round(dihedral(A, B, C, D), 3), "degrees")

7) Contact maps and local packing

analysis

A contact map turns a 3D structure into a 2D matrix. Off-diagonal contacts often reveal long-range interactions, domain interfaces, and β-sheet patterns.

\[ C_{ij} = \begin{cases} 1 & \text{if } \|\mathbf x_i - \mathbf x_j\| < r_c \\ 0 & \text{otherwise} \end{cases} \]

Why it helps

Contact maps are compact, easy to cluster, and useful for comparing variants or homologs.

Real-life use

If a mutation removes long-range packing between two regions, a contact map often shows the effect immediately.

Annotated Python: build a simple Cα contact map 
import numpy as np
import matplotlib.pyplot as plt
from Bio.PDB import MMCIFParser

pdb_file = "4HHB.cif"
chain_id = "A"
cutoff = 8.0

parser = MMCIFParser(QUIET=True)
structure = parser.get_structure("4HHB", pdb_file)
chain = structure[0][chain_id]

coords = []
for res in chain:
    if res.id[0] == " " and "CA" in res:
        coords.append(res["CA"].coord)

coords = np.array(coords)
dist = np.sqrt(((coords[:, None, :] - coords[None, :, :]) ** 2).sum(axis=2))
contacts = (dist < cutoff).astype(int)

plt.figure(figsize=(6, 5))
plt.imshow(contacts, origin="lower", interpolation="nearest")
plt.title(f"Cα contact map: 4HHB chain {chain_id}")
plt.xlabel("Residue index")
plt.ylabel("Residue index")
plt.tight_layout()
plt.show()

8) Optimal superposition and RMSD

comparison

Two structures cannot be compared directly in raw coordinates because they may be shifted or rotated. First superpose them optimally. Then compute RMSD.

\[ \mathrm{RMSD} = \sqrt{\frac{1}{N}\sum_{i=1}^{N}\|\mathbf x_i - \mathbf y_i\|^2} \]
The Kabsch algorithm finds the rotation matrix that minimizes the squared error between matched atoms.
RMSD is intuitive but sensitive to outliers. One flexible loop or hinge motion can inflate it even when the global fold is conserved.
Annotated Python: Kabsch alignment and RMSD 
import numpy as np

def kabsch(P, Q):
    P = np.asarray(P, float)
    Q = np.asarray(Q, float)
    Pc = P - P.mean(axis=0)
    Qc = Q - Q.mean(axis=0)
    C = Pc.T @ Qc
    V, S, Wt = np.linalg.svd(C)
    d = np.sign(np.linalg.det(V @ Wt))
    D = np.diag([1.0, 1.0, d])
    U = V @ D @ Wt
    P_rot = Pc @ U
    rmsd = np.sqrt(np.mean(np.sum((P_rot - Qc) ** 2, axis=1)))
    return U, P.mean(axis=0), Q.mean(axis=0), rmsd

P = np.array([[0,0,0],[1,0,0],[0,1,0],[0,0,1]], dtype=float)
R = np.array([[0,-1,0],[1,0,0],[0,0,1]], dtype=float)
t = np.array([5.0, 2.0, -1.0])
Q = P @ R.T + t

U, centroid_P, centroid_Q, rmsd = kabsch(P, Q)
print("RMSD after optimal superposition:", rmsd)

9) TM-score and structural alignment

advanced comparison

TM-score was designed to capture global topological similarity in a way that is less dominated by local outliers and less size-dependent than RMSD.

A common form is \[ \mathrm{TM\text{-}score} = \max \left[ \frac{1}{L_\text{target}} \sum_{i=1}^{L_\text{aligned}} \frac{1}{1 + (d_i / d_0(L_\text{target}))^2} \right] \]
MetricBest useMain weakness
RMSDLocal matched-atom comparisonDominated by outliers
TM-scoreGlobal fold similarityNeeds a structural alignment
CoverageHow much actually alignsHigh coverage does not guarantee high-quality local geometry

10) Secondary structure, exposure, and domains

interpretation

Beyond coordinates, you usually want to know whether a residue sits in a helix, sheet, loop, buried core, solvent-exposed surface, or chain interface.

Secondary structure

Helices and sheets arise from repeating local hydrogen-bond patterns. Many functionally critical regions, however, are irregular loops.

Solvent exposure

Buried substitutions often affect stability more strongly than surface substitutions, while exposed substitutions may alter interaction specificity.

A useful triage rule for variants is: buried + drastic physicochemical change + high-confidence region deserves close attention.

12) Homology modeling logic

classic workflow

Classical modeling follows the logic: template search → sequence alignment → model building → validation. Even in the era of end-to-end deep predictors, this mindset still matters.

StepWhy it mattersTypical failure
Template searchGets the right fold/stateWrong conformational state or distant template
AlignmentDefines residue correspondenceBad gap placement near functional loops
ModelingPlaces backbone and side chainsClashes or unrealistic loops
ValidationRejects poor geometryUsing the model without quality checks

13) AlphaFold-style prediction and confidence

prediction

Modern predictors produce one or more structures plus confidence information. The key skill is learning which regions are reliable and which are not.

pLDDT

Per-residue confidence. High values often indicate reliable local geometry; low values often indicate flexible, disordered, or uncertain regions.

PAE

Pairwise or domain-level uncertainty. A model can have good local geometry but uncertain inter-domain placement.

Do not treat the whole model as uniformly trustworthy. Evaluate confidence region by region.

import numpy as np

16) Complexes, interfaces, ligands, and uncertainty

complex biology

Monomers are usually easier than complexes. Once you add protein-protein interfaces, nucleic acids, ligands, ions, or membrane context, several plausible states may exist.

QuestionWhat to check
Does this dimer look real?Interface confidence, buried area, conservation, experimental support
Does this mutation alter binding?Interface chemistry, side-chain packing, electrostatics, ΔΔG

17) Mutation analysis and ΔΔG

variants

A common computational question is whether a mutation changes structural stability. The standard quantity is ΔΔG.

\[ \Delta\Delta G = \Delta G_{\text{mutant}} - \Delta G_{\text{wild type}} \]

Positive ΔΔG is often interpreted as destabilizing, but always confirm the software sign convention.

Interpretation

Buried mutations that increase steric clash or bury charge often raise predicted destabilization.

Caveat

Not every harmful mutation acts through folding stability. Some primarily disrupt binding, catalysis, or dynamics.

Annotated shell: FoldX mutation workflow 
#!/usr/bin/env bash
set -euo pipefail

PDB="input_structure.pdb"
MUT="A123V"
CHAIN="A"

echo "${CHAIN}${MUT};" > individual_list.txt
foldx --command=RepairPDB --pdb="${PDB}"
foldx --command=BuildModel \
      --pdb="${PDB%.pdb}_Repair.pdb" \
      --mutant-file=individual_list.txt \
      --numberOfRuns=5

19) Molecular dynamics fundamentals

simulation

MD simulates atomic motion through time under a force field. It is useful when one static structure is not enough: pocket opening, loop motion, domain shifts, or mutation-dependent flexibility.

Lennard–Jones: \[ V_{LJ}(r) = 4\epsilon \left[\left(\frac{\sigma}{r}\right)^{12} - \left(\frac{\sigma}{r}\right)^6\right] \]
Coulomb: \[ V_{elec}(r) = \frac{k q_i q_j}{\varepsilon r} \]
Conformations are sampled approximately with Boltzmann weighting: \[ p_i \propto e^{-E_i / (k_B T)} \]
Annotated Python: minimal OpenMM setup 
from openmm.app import *
from openmm import *
from openmm.unit import *

pdb = PDBFile("input_structure.pdb")
forcefield = ForceField("amber14-all.xml", "amber14/tip3pfb.xml")

modeller = Modeller(pdb.topology, pdb.positions)
modeller.addHydrogens(forcefield)
modeller.addSolvent(forcefield, padding=1.0*nanometer)

system = forcefield.createSystem(
    modeller.topology,
    nonbondedMethod=PME,
    nonbondedCutoff=1.0*nanometer,
    constraints=HBonds
)

integrator = LangevinMiddleIntegrator(300*kelvin, 1/picosecond, 0.004*picoseconds)
simulation = Simulation(modeller.topology, system, integrator)
simulation.context.setPositions(modeller.positions)
simulation.minimizeEnergy()
simulation.reporters.append(DCDReporter("traj.dcd", 1000))
simulation.step(25000)

20) RMSD and RMSF on trajectories

simulation analysis

After a trajectory, the first questions are: did the system drift, and which residues are flexible? RMSD and RMSF are standard first-pass answers.

\[ \mathrm{RMSF}_i = \sqrt{\left\langle \|\mathbf x_i(t) - \langle \mathbf x_i \rangle\|^2 \right\rangle_t} \]
Annotated Python: RMSD and RMSF with MDAnalysis 
import MDAnalysis as mda
from MDAnalysis.analysis import rms
from MDAnalysis.analysis.rms import RMSF
import matplotlib.pyplot as plt

u = mda.Universe("final.pdb", "traj.dcd")
R = rms.RMSD(u, u, select="protein and backbone", ref_frame=0)
R.run()

plt.figure(figsize=(6, 4))
plt.plot(R.results.rmsd[:, 1], R.results.rmsd[:, 2])
plt.xlabel("Frame")
plt.ylabel("Backbone RMSD (Å)")
plt.tight_layout()
plt.show()

protein = u.select_atoms("protein and name CA")
rmsf = RMSF(protein).run()

plt.figure(figsize=(7, 4))
plt.plot(protein.resids, rmsf.results.rmsf)
plt.xlabel("Residue number")
plt.ylabel("Cα RMSF (Å)")
plt.tight_layout()
plt.show()
A rising RMSD is not automatically “bad.” The protein may have moved to another stable state rather than unfolded.

22) Pairwise similarity matrices and clustering

family view

Once you have pairwise TM-scores or Foldseek-based similarities, turn them into a matrix and cluster them. This is often the clearest summary when comparing many homologs or many mutant structures.

A simple distance transformation is \[ D_{ij} = 1 - S_{ij} \] where \( S_{ij} \) is a similarity measure such as TM-score.
Annotated Python: clustered heatmap from a similarity matrix 
import numpy as np
import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

proteins = ["WT", "Mut1", "Mut2", "HomologA", "HomologB"]
tm = np.array([
    [1.00, 0.93, 0.89, 0.55, 0.51],
    [0.93, 1.00, 0.87, 0.53, 0.50],
    [0.89, 0.87, 1.00, 0.57, 0.54],
    [0.55, 0.53, 0.57, 1.00, 0.84],
    [0.51, 0.50, 0.54, 0.84, 1.00],
])

df = pd.DataFrame(tm, index=proteins, columns=proteins)
sns.clustermap(df, cmap="viridis", annot=True, fmt=".2f")
plt.show()

23) Real-life case-study logic

application

p53-style variant

Ask whether the mutation destabilizes the fold, disrupts DNA contact, or changes dynamics.

AMR porin / enzyme

Ask whether the mutation changes pore size, electrostatics, catalytic geometry, or inhibitor binding.

Receptor-binding domain

Ask whether the mutation is interfacial, second-shell, or allosteric.

General rule

Use structure to formulate a mechanism, then test it with confidence metrics, comparison, and if possible phenotype data.
