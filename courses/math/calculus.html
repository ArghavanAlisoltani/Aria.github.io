3) Derivatives

Beginner → Intermediate
\[ f'(x)=\lim_{h\to 0}\frac{f(x+h)-f(x)}{h} \]
Example 3.1: \(f(x)=x^2\Rightarrow f'(x)=2x\) from definition
\[ \frac{(x+h)^2-x^2}{h}=\frac{2xh+h^2}{h}=2x+h\Rightarrow f'(x)=2x \]
Python: numeric derivative vs exact
Example 3.2: product rule — \(f(x)=x^2e^x\)
\[ (x^2e^x)'=2xe^x+x^2e^x=e^x(2x+x^2) \]
Python: symbolic + plot

7) Integration techniques

Intermediate
Example 7.1: substitution \(\int 2x\cos(x^2)dx=\sin(x^2)+C\)
\[ u=x^2,\ du=2x\,dx \Rightarrow \int 2x\cos(x^2)dx=\int \cos u\,du=\sin u + C \]
Python: differentiate check
Example 7.2: by parts \(\int xe^x dx = e^x(x-1)+C\)
\[ \int u\,dv=uv-\int v\,du,\quad u=x,\ dv=e^x dx \]
Python: symbolic check + plot

8) Improper integrals

Intermediate → Advanced
Example 8.1: \(\int_1^\infty 1/x^p\,dx\)
\[ \int_1^\infty x^{-p}dx= \lim_{b\to\infty}\left[\frac{x^{1-p}}{1-p}\right]_1^b \]
Converges if \(p>1\) to \(1/(p-1)\), diverges if \(p\le 1\).
Python: numeric approximation

16) Practice set

Exercises
Problems
  1. \(\lim_{x\to 0}\frac{1-\cos x}{x^2}\)
  2. Differentiate \(f(x)=\ln(1+x^2)\)
  3. Classify critical points of \(f(x)=x^3-3x\)
  4. Compute \(\int x\ln x\,dx\)
  5. Approximate \(\sum_{n=1}^{\infty}\frac{1}{n^2}\) with partial sums
Answer outlines
  • \(\to 1/2\)
  • \(f'(x)=2x/(1+x^2)\)
  • \(x=\pm1\); use \(f''(x)=6x\)
  • By parts with \(u=\ln x\), \(dv=x\,dx\)
  • Converges to \(\pi^2/6\)
