20 Distance Methods

A compact course explaining 20 different distance (and distance-like) methods with: (1) mathematical definition, (2) a small numerical example, (3) Python code, and (4) a Matplotlib plot snippet.

Important notes

Some measures are true metrics (satisfy triangle inequality), others are distance-like divergences (useful but not metrics). Each section states what it is used for.

Re-used across methods

1) Euclidean distance (L2)

Metric
\[ d_2(\mathbf{x},\mathbf{y})=\|\mathbf{x}-\mathbf{y}\|_2=\sqrt{\sum_{i=1}^n (x_i-y_i)^2} \]
Numerical example
\[ \mathbf{x}-\mathbf{y}=(1-2,\ 2-0,\ 3-4)=(-1,2,-1) \] \[ d_2=\sqrt{(-1)^2+2^2+(-1)^2}=\sqrt{1+4+1}=\sqrt{6}\approx 2.449 \]
Python + plot (distance matrix heatmap)

2) Manhattan distance (L1)

Metric
\[ d_1(\mathbf{x},\mathbf{y})=\|\mathbf{x}-\mathbf{y}\|_1=\sum_{i=1}^n |x_i-y_i| \]
Numerical example
\[ d_1=|1-2|+|2-0|+|3-4|=1+2+1=4 \]
Python + plot

3) Chebyshev distance (L∞)

Metric
\[ d_\infty(\mathbf{x},\mathbf{y})=\max_i |x_i-y_i| \]
Numerical example
\[ |x-y|=(1,2,1)\Rightarrow d_\infty=\max(1,2,1)=2 \]
Python + plot

4) Minkowski distance (p = 3)

Metric (p ≥ 1)
\[ d_p(\mathbf{x},\mathbf{y})=\left(\sum_i |x_i-y_i|^p\right)^{1/p} \]
Numerical example (p=3)
\[ d_3=\left(|-1|^3+|2|^3+|-1|^3\right)^{1/3} =(1+8+1)^{1/3}=10^{1/3}\approx 2.154 \]
Python + plot

5) Mahalanobis distance

Metric (if Σ is PD)

Mahalanobis accounts for feature scale and correlations using a covariance matrix \(\Sigma\). Common in multivariate outlier detection and Gaussian models.

\[ d_M(\mathbf{x},\mathbf{y})=\sqrt{(\mathbf{x}-\mathbf{y})^T \Sigma^{-1} (\mathbf{x}-\mathbf{y})} \]
Numerical example (simple diagonal Σ)

Let \(\Sigma=\mathrm{diag}(1,4,1)\Rightarrow \Sigma^{-1}=\mathrm{diag}(1,1/4,1)\). For \(\Delta=(-1,2,-1)\):

\[ \Delta^T\Sigma^{-1}\Delta = (-1)^2\cdot 1 + 2^2\cdot\frac{1}{4} + (-1)^2\cdot 1 =1+1+1=3 \] \[ d_M=\sqrt{3}\approx 1.732 \]
Python + plot

6) Cosine distance

Distance-like (metric variant exists)

Cosine measures orientation difference rather than magnitude. Often used for TF-IDF vectors, embeddings, gene expression signatures (after normalization).

\[ \mathrm{cos\_sim}(\mathbf{x},\mathbf{y})=\frac{\mathbf{x}^T\mathbf{y}}{\|\mathbf{x}\|\|\mathbf{y}\|}, \quad d_{\cos}=1-\mathrm{cos\_sim} \]
Numerical example
\[ \mathbf{x}^T\mathbf{y}=1\cdot2+2\cdot0+3\cdot4=14 \] \[ \|\mathbf{x}\|=\sqrt{14},\ \|\mathbf{y}\|=\sqrt{20} \Rightarrow \mathrm{cos\_sim}=\frac{14}{\sqrt{14}\sqrt{20}}=\frac{14}{\sqrt{280}}\approx 0.837 \] \[ d_{\cos}\approx 1-0.837=0.163 \]
Python + plot

7) Pearson correlation distance

Distance-like
\[ \rho(\mathbf{x},\mathbf{y})=\frac{\sum_i (x_i-\bar{x})(y_i-\bar{y})}{\sqrt{\sum_i(x_i-\bar{x})^2}\sqrt{\sum_i(y_i-\bar{y})^2}}, \quad d=1-\rho \]
Numerical example (brief)

Compute centered vectors \(\tilde{\mathbf{x}}=\mathbf{x}-\bar{x}\), \(\tilde{\mathbf{y}}=\mathbf{y}-\bar{y}\), then cosine similarity of centered vectors equals Pearson correlation.

Python + plot

8) Spearman correlation distance (rank-based)

Distance-like

Spearman is Pearson correlation applied to ranks. Robust to monotonic but nonlinear relationships.

\[ d = 1 - \rho(\mathrm{rank}(\mathbf{x}),\ \mathrm{rank}(\mathbf{y})) \]
Numerical example

Rank the entries of x and y, then compute Pearson correlation on ranks.

Python + plot

9) Hamming distance (binary vectors or equal-length strings)

Metric
\[ d_H(\mathbf{a},\mathbf{b})=\sum_i \mathbb{1}[a_i\ne b_i] \]
Numerical example (binary)
\[ \mathbf{a}=(1,0,1,1),\ \mathbf{b}=(1,1,0,1) \Rightarrow d_H = 2 \]
Python + plot

10) Jaccard distance (binary sets)

Metric (for sets)
\[ J(A,B)=\frac{|A\cap B|}{|A\cup B|},\quad d_J=1-J(A,B) \]
Numerical example

Let \(A=\{1,3,4\}\), \(B=\{1,2,4\}\). Then \(|A\cap B|=2\), \(|A\cup B|=4\).

\[ J=2/4=0.5,\quad d_J=0.5 \]
Python + plot

11) Dice / Sørensen distance

Distance-like
\[ S(A,B)=\frac{2|A\cap B|}{|A|+|B|},\quad d=1-S(A,B) \]
Numerical example

With \(A=\{1,3,4\}\), \(B=\{1,2,4\}\): \(|A|=3,|B|=3,|A\cap B|=2\).

\[ S=\frac{2\cdot 2}{3+3}=\frac{4}{6}=\frac{2}{3}\approx 0.667,\quad d\approx 0.333 \]
Python + plot

12) Bray–Curtis distance (counts/composition)

Distance-like

Common in ecology/microbiome for abundance vectors (nonnegative).

\[ d_{BC}(\mathbf{x},\mathbf{y})= \frac{\sum_i |x_i-y_i|}{\sum_i (x_i+y_i)} \]
Numerical example (nonnegative counts)

Let \(\mathbf{x}=(1,2,3)\), \(\mathbf{y}=(2,0,4)\).

\[ \sum|x_i-y_i|=4,\quad \sum(x_i+y_i)= (1+2)+(2+0)+(3+4)=12 \Rightarrow d_{BC}=4/12=0.333 \]
Python + plot

13) Canberra distance

Metric (for nonnegative with care)
\[ d_{Can}(\mathbf{x},\mathbf{y})=\sum_i \frac{|x_i-y_i|}{|x_i|+|y_i|} \] (treat 0/0 terms as 0)
Numerical example
\[ \frac{|1-2|}{|1|+|2|}=\frac{1}{3},\quad \frac{|2-0|}{|2|+|0|}=\frac{2}{2}=1,\quad \frac{|3-4|}{|3|+|4|}=\frac{1}{7} \] \[ d_{Can}=\frac{1}{3}+1+\frac{1}{7}\approx 1.476 \]
Python + plot

14) Chi-square distance (histograms)

Distance-like

Often used for comparing histograms (nonnegative). A common symmetric form:

\[ d_{\chi^2}(\mathbf{x},\mathbf{y})= \frac{1}{2}\sum_i \frac{(x_i-y_i)^2}{x_i+y_i} \] (treat 0/0 as 0)
Numerical example

Using \(\mathbf{x}=(1,2,3)\), \(\mathbf{y}=(2,0,4)\):

\[ \frac{1}{2}\left( \frac{(-1)^2}{3}+\frac{(2)^2}{2}+\frac{(-1)^2}{7} \right) =\frac{1}{2}\left(\frac{1}{3}+2+\frac{1}{7}\right)\approx 1.238 \]
Python + plot

15) Hellinger distance (probability distributions)

Metric
\[ H(\mathbf{p},\mathbf{q})=\frac{1}{\sqrt{2}}\left\|\sqrt{\mathbf{p}}-\sqrt{\mathbf{q}}\right\|_2 =\sqrt{1-\sum_i \sqrt{p_i q_i}} \]
Numerical example

Let \(p=(0.5,0.5)\), \(q=(0.9,0.1)\):

\[ \sum \sqrt{p_i q_i}=\sqrt{0.45}+\sqrt{0.05}\approx 0.671+0.224=0.895 \] \[ H=\sqrt{1-0.895}\approx 0.324 \]
Python + plot (bar charts for p and q)

16) Jensen–Shannon distance (JS)

Metric (sqrt(JS))

JS divergence is symmetric and finite (unlike KL). The square root of JS divergence is a metric.

\[ m=\frac{1}{2}(p+q),\quad JS(p,q)=\frac{1}{2}KL(p\|m)+\frac{1}{2}KL(q\|m) \] \[ d_{JS}(p,q)=\sqrt{JS(p,q)} \]
Numerical example (concept)

Use the same \(p=(0.5,0.5)\), \(q=(0.9,0.1)\); compute \(m=(0.7,0.3)\); then KL terms.

Python + plot

17) Bhattacharyya distance

Distance-like
\[ BC(p,q)=\sum_i \sqrt{p_i q_i},\quad D_B(p,q)=-\ln(BC(p,q)) \]
Numerical example

Using the previous coefficient \(BC\approx 0.895\):

\[ D_B=-\ln(0.895)\approx 0.111 \]
Python + plot

18) Wasserstein-1 distance (Earth Mover, 1D)

Metric (for distributions)

For 1D distributions, Wasserstein-1 equals the area between CDFs (intuitive “work” to move mass).

\[ W_1(P,Q)=\int_{-\infty}^{\infty} |F_P(t)-F_Q(t)|\,dt \]
Numerical example (discrete samples)

Example: samples A=[0,1,2], B=[0,2,4] (same size). Compute with SciPy if available.

Python + plot (CDFs)

19) Dynamic Time Warping (DTW)

Distance-like (not a metric in general)

DTW aligns two sequences allowing “warping” in time. Very useful for signals with phase shifts.

\[ DTW(A,B)=\min_{\pi}\sum_{(i,j)\in \pi} |A_i-B_j| \] where \(\pi\) is a valid warping path.
Numerical example

A=[1,2,3,4], B=[1,1,2,3,4] → DTW can align repeated 1 in B with 1 in A.

Python + plot (cost matrix)

20) Levenshtein edit distance (strings)

Metric

Minimum number of single-character insertions, deletions, substitutions to transform one string into another. Common for sequence strings (toy-scale), barcodes, and approximate matching.

\[ d_{Lev}(s,t)=\min \#(\text{ins, del, sub}) \]
Numerical example

s="GATTACA", t="GCATGCU" → distance can be computed by DP (classic example).

Python + plot (DP table heatmap)
