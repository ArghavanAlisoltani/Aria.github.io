1) What is regression? What is regularization?

Core concept

Regression tries to learn a function \(f(x)\) that predicts \(y\) from \(x\). In linear regression, \(f(x)=\beta_0+\beta_1 x\).

OLS objective (no regularization)
\[ \hat{\beta}=\arg\min_{\beta}\ \sum_{i=1}^n (y_i - \mathbf{x}_i^T\beta)^2 =\arg\min_{\beta}\ \|y - X\beta\|_2^2 \]
Regularized objective (Ridge / Lasso / Elastic Net)
\[ \hat{\beta}=\arg\min_{\beta}\ \|y - X\beta\|_2^2 + \lambda \,\Omega(\beta) \] where: \[ \Omega(\beta)=\|\beta\|_2^2 \text{ (Ridge)},\quad \Omega(\beta)=\|\beta\|_1 \text{ (Lasso)},\quad \Omega(\beta)=\alpha\|\beta\|_1+(1-\alpha)\|\beta\|_2^2 \text{ (Elastic Net)} \]
Regularization introduces bias but reduces variance, often improving test performance (especially when features are correlated or when \(p\) is large relative to \(n\)).
Numerical mini-example (why penalties shrink coefficients)

Suppose a 1D model has a coefficient \(\beta=10\). Ridge adds \(\lambda \beta^2\) to the loss. If \(\lambda=1\), then just the penalty adds 100 units, so the optimizer prefers smaller \(\beta\) unless it greatly reduces SSE.

2) OLS Linear Regression

Baseline
\[ y = X\beta + \varepsilon,\quad \hat{\beta}_{OLS} = (X^TX)^{-1}X^Ty \]
Numerical example (tiny)

For points (x,y) = (0,1), (1,2), (2,2), the best-fit line is \(\hat{y}=\frac{7}{6}+\frac{1}{2}x\). (Shown previously in your linear algebra course.)

3) Polynomial Regression (linear in parameters)

Flexible

Polynomial regression expands features: \(x \mapsto [1,x,x^2,\dots,x^d]\). It’s still linear regression in the transformed features.

\[ \hat{y}=\beta_0+\beta_1 x+\beta_2 x^2+\cdots+\beta_d x^d \]
Python + plot (degree 1 vs 5)
Higher-degree models can overfit; regularization (Ridge/Lasso) is commonly used with polynomial features.

4) Ridge Regression (L2 regularization)

Penalizes large weights
\[ \hat{\beta}_{ridge}=\arg\min_{\beta}\ \|y-X\beta\|_2^2 + \lambda\|\beta\|_2^2 \] Closed-form: \[ \hat{\beta}_{ridge}=(X^TX+\lambda I)^{-1}X^Ty \]
Python + plot (OLS vs Ridge with polynomial features)

5) Lasso Regression (L1 regularization)

Sparse coefficients
\[ \hat{\beta}_{lasso}=\arg\min_{\beta}\ \|y-X\beta\|_2^2 + \lambda\|\beta\|_1 \]

L1 penalty creates sparsity (some coefficients exactly 0) → can behave like feature selection.

Python + plot (coefficients shrink to zero)

6) Elastic Net (L1 + L2)

Best of both
\[ \hat{\beta}=\arg\min_{\beta}\ \|y-X\beta\|_2^2 + \lambda\left(\alpha\|\beta\|_1+(1-\alpha)\|\beta\|_2^2\right) \]

Helpful when predictors are correlated: Lasso may arbitrarily pick one; Elastic Net tends to keep groups.

Python + plot (compare Lasso vs ElasticNet)

7) Logistic Regression (classification, part of GLM family)

Binary outcomes

Logistic regression models \(P(y=1|x)\) via the sigmoid: \(\sigma(z)=1/(1+e^{-z})\).

\[ P(y=1|\mathbf{x})=\sigma(\mathbf{x}^T\beta),\quad \text{Loss}=-\sum_i \left[y_i\log p_i + (1-y_i)\log(1-p_i)\right] + \lambda\Omega(\beta) \]
Numerical example

If \(\beta_0=-2\), \(\beta_1=0.8\), and \(x=3\), then \(z=-2+0.8(3)=0.4\), so \(p=\sigma(0.4)\approx 0.599\).

Python + plot (decision boundary)

8) Poisson Regression (count data)

GLM for counts

Poisson regression assumes \(y_i \sim \mathrm{Poisson}(\mu_i)\) with log link: \(\log \mu_i = \mathbf{x}_i^T\beta\).

\[ y_i\sim \mathrm{Poisson}(\mu_i),\quad \log \mu_i = \mathbf{x}_i^T\beta \]
Numerical example

If \(\log \mu = 1.2\), then \(\mu=e^{1.2}\approx 3.32\). Expected count ≈ 3.32.

Python + plot (fitted mean curve)

9) Negative Binomial Regression (overdispersed counts)

Counts with variance > mean

When counts have variance larger than mean (common in RNA-seq), NB is often better than Poisson.

\[ y_i \sim \mathrm{NB}(\mu_i, \theta),\quad \log \mu_i = \mathbf{x}_i^T\beta \]
Python + plot (Poisson vs NB fit)

10) Robust Regression (Huber)

Outlier-resistant

Huber regression uses a loss that is quadratic for small residuals and linear for large residuals. It reduces the impact of outliers compared to OLS.

\[ L_\delta(r)= \begin{cases} \frac{1}{2}r^2 & |r|\le \delta\\ \delta(|r|-\frac{1}{2}\delta) & |r|>\delta \end{cases} \]
Python + plot (OLS vs Huber with outliers)

11) Quantile Regression (median regression)

Robust to asymmetric noise

Quantile regression fits conditional quantiles (e.g., median) rather than conditional mean. For the median (\(\tau=0.5\)), it minimizes absolute deviations.

\[ \min_{\beta}\sum_i \rho_\tau(y_i-\mathbf{x}_i^T\beta), \quad \rho_\tau(u)=u(\tau-\mathbb{1}[u<0]) \]
Python + plot (mean vs median fit)
