What is a protein structure?

Core idea

A protein structure is a 3D arrangement of atoms. Every atom has coordinates: x, y, and z. Those coordinates let you measure distances, angles, contacts, and active-site geometry.

  • Residue: one amino acid in the chain
  • Atom: individual coordinate point like N, CA, C, O
  • Chain: one polypeptide in the structure
  • Ligand: a bound small molecule

Three levels to remember first

  • Primary structure: amino-acid sequence
  • Secondary structure: helices, sheets, loops
  • Tertiary structure: full 3D fold of one chain

Later you will also hear quaternary structure, which means multiple chains assembled together.

Slide 3

Amino acids and side chains

Nonpolar vs Polar

A nonpolar amino acid is one with a side chain that has no charge and an even distribution of electrons. Instead of being "water-loving" (hydrophilic), they are hydrophobic (water-fearing). A polar amino acid is one that has a "dipole"—meaning its side chain (R-group) has an uneven distribution of electrons. This creates a partial positive charge on one end and a partial negative charge on the other, making them "water-loving" (hydrophilic).

Polar uncharged

These have side chains that are polar but stay electrically neutral. Asparagine (Asn) & Glutamine (Gln): These contain an amide group (NH2-C=O). While the oxygen and nitrogen create polarity, the group does not pick up or lose an H+ ion in the body. They are "peaceful" polar molecules.

Charged

These side chains are polar enough to actually ionize (lose a proton), giving them a full negative charge. Aspartic Acid (Asp) & Glutamic Acid (Glu): These contain a carboxylic acid group (COOH). At body pH, they lose the H and become Aspartate and Glutamate, carrying a net -1 charge.

Slide 4

Amino acids and side chains

Nonpolar

Gly, Ala, Val, Leu, Ile, Met, Pro, Phe, Trp The side chains of these amino acids are mostly made of carbon and hydrogen (hydrocarbons):

-Aliphatic (Straight chains): Glycine (the simplest), Alanine, Valine, Leucine, and Isoleucine.

-Aromatic (Ring structures): Phenylalanine and Tryptophan (these are bulky and "greasy").

-Sulfur-containing: Methionine.

-The Weird One: Proline, which loops back and bonds to its own backbone, creating a rigid kink in the protein chain.

Polar uncharged

Ser, Thr, Asn, Gln, Tyr, Cys

Charged

Asp, Glu, Lys, Arg, His

In active sites, the side chain matters more than the residue name alone. Asp/Glu often donate or accept protons through charged oxygen atoms. Lys/Arg often stabilize negative charge. Tyr/Ser/Cys can participate in hydrogen bonding or chemistry.

Slide 5

Primary, secondary, tertiary structure

Primary

The linear amino-acid sequence.

Secondary

Local patterns such as α-helices and β-sheets, stabilized largely by backbone hydrogen bonds.

Tertiary

The complete 3D arrangement of one chain, including packing of helices, sheets, loops, and side chains.

Examples of 3D Structures: Pseudomonas aeruginosa

1) OprD outer membrane channel

OprD is a beta-barrel outer-membrane channel whose structure explains how a protein can act as a selective passageway, using pore shape and chemically important residues to favor transport of certain substrates, especially carboxylate-containing molecules.

PDB: 2ODJ, 3SY7

  • OprD: forms a β-barrel architecture typical of outer-membrane porins the structure contains a central pore
  • What to learn: beta-barrel fold, pore constriction, charged surfaces, channel selectivity.
  • Look for: the pore interior, charged residues, and how the channel differs from an enzyme active site.
  • Good beginner question: Where is the narrowest region of the channel and which residues shape it?

OprD is a strong example for understanding that not all biologically important structures are classic enzymes.

# Optional beginner Python idea:
# After loading a PDB with Bio.PDB, find residues within 4.0 Å of a ligand
# and print their residue name, number, chain, and minimum atom distance.

from Bio.PDB import PDBParser

pdb_file = "example.pdb"
ligand_name = "HSL"   # replace with actual residue name from the structure
cutoff = 4.0

parser = PDBParser(QUIET=True)
structure = parser.get_structure("protein", pdb_file)

ligand_atoms = []
protein_residues = []

for model in structure:
    for chain in model:
        for residue in chain:
            hetflag = residue.id[0]
            if residue.resname == ligand_name:
                ligand_atoms.extend(list(residue.get_atoms()))
            elif hetflag == " ":
                protein_residues.append((chain.id, residue))

for chain_id, residue in protein_residues:
    min_dist = None
    for atom in residue.get_atoms():
        for latom in ligand_atoms:
            d = atom - latom
            if min_dist is None or d < min_dist:
                min_dist = d
    if min_dist is not None and min_dist <= cutoff:
        print(chain_id, residue.resname, residue.id[1], round(min_dist, 2))

Tip: For true beginners, use LasR → FabA → OprD in that order.

Slide 10

Enzymes and active sites

An enzyme is a protein that speeds up a chemical reaction. The active site is the local region where substrate binding and chemistry happen.

  • Catalytic residues are often conserved
  • Active sites are usually pockets or clefts
  • Ligands often mark the active site in a crystal structure

What to look for visually

  • Bound substrate or inhibitor
  • Nearby charged or polar residues
  • Metal ions or cofactors
  • Clusters of H-bonds around the ligand
# Python example: find residues near a ligand. # This helps you identify a likely active site from structure coordinates. from Bio.PDB import PDBParser, NeighborSearch, Selection pdb_file = "6u4i.pdb" ligand_name = "EQU" # example ligand in 6U4I; change if needed cutoff = 4.0 # angstroms parser = PDBParser(QUIET=True) structure = parser.get_structure("ksi", pdb_file) model = structure[0] # Gather all atoms for neighbor searching all_atoms = list(structure.get_atoms()) ns = NeighborSearch(all_atoms) # Find ligand atoms ligand_atoms = [] for residue in structure.get_residues(): hetflag = residue.id[0] if hetflag.startswith("H_") and residue.get_resname().strip() == ligand_name: ligand_atoms.extend(list(residue.get_atoms())) print(f"Found {len(ligand_atoms)} ligand atoms") # Search nearby residues around each ligand atom nearby_residues = set() for atom in ligand_atoms: for neighbor in ns.search(atom.coord, cutoff, level="R"): if neighbor.get_resname() != ligand_name: nearby_residues.add((neighbor.get_parent().id, neighbor.get_resname(), neighbor.id[1])) for chain_id, resname, resnum in sorted(nearby_residues): print(chain_id, resname, resnum) # Interpretation: # residues repeatedly close to the ligand are often part of the binding pocket # and may include catalytic residues.
Slide 6

Ligands, cofactors, metal ions

Ligand

A bound small molecule such as inhibitor, substrate, or product analog.

Cofactor

A non-protein helper like NAD, FAD, heme, PLP, biotin.

Metal ion

Zn, Mg, Fe, Mn, Ca and others that can stabilize charge or participate directly in catalysis.

# Python example: list hetero residues, waters, and common ions in a structure. # This is one of the first things to do when reading a PDB file. from Bio.PDB import PDBParser COMMON_IONS = {"ZN", "MG", "FE", "MN", "CA", "NA", "K", "CU", "CO", "NI"} parser = PDBParser(QUIET=True) structure = parser.get_structure("example", "6u4i.pdb") ligands = [] waters = [] ions = [] for residue in structure.get_residues(): hetflag = residue.id[0] # ' ' for standard residues, 'W' for water, 'H_xxx' for hetero resname = residue.get_resname().strip() chain_id = residue.get_parent().id resnum = residue.id[1] if hetflag == "W": waters.append((chain_id, resname, resnum)) elif resname in COMMON_IONS: ions.append((chain_id, resname, resnum)) elif hetflag.startswith("H_"): ligands.append((chain_id, resname, resnum)) print("Ligands:", ligands[:10]) print("Waters:", waters[:10], "... total", len(waters)) print("Ions:", ions)
Slide 7

Hydrogen bonds, salt bridges, hydrophobic contacts

Very simple heuristics

  • Hydrogen bond: donor/acceptor atoms often within about 2.6–3.5 Å
  • Salt bridge: opposite charges, often within about 4 Å
  • Hydrophobic contact: nonpolar atoms packed together, often 3.5–5 Å

These are rough educational cutoffs, not full chemical definitions.

# Python example: measure distances from a ligand oxygen to nearby atoms. # Start simple: distance is the first structural quantity every beginner should trust. from Bio.PDB import PDBParser import numpy as np parser = PDBParser(QUIET=True) structure = parser.get_structure("ksi", "6u4i.pdb") # Find a target ligand atom, for example the first oxygen atom in ligand EQU ligand_atom = None for residue in structure.get_residues(): if residue.get_resname().strip() == "EQU": for atom in residue: if atom.element == "O": ligand_atom = atom break if ligand_atom: break if ligand_atom is None: raise ValueError("No ligand oxygen atom found") print("Target atom:", ligand_atom.get_full_id()) # Compute distances to all protein atoms and report the closest 15 hits = [] for atom in structure.get_atoms(): parent_residue = atom.get_parent() if parent_residue.get_resname().strip() == "EQU": continue d = np.linalg.norm(atom.coord - ligand_atom.coord) hits.append((d, atom.get_parent().get_parent().id, parent_residue.get_resname(), parent_residue.id[1], atom.name)) for d, chain_id, resname, resnum, atom_name in sorted(hits)[:15]: print(f"{d:5.2f} Å {chain_id} {resname} {resnum} {atom_name}") # Interpretation: # close N/O atoms may suggest hydrogen bonds # close Lys/Asp/Glu/Arg side-chain atoms may suggest electrostatic interactions.

PDB files and what coordinates mean

A PDB file stores atom-level information. Each atom line includes atom name, residue name, chain ID, residue number, 3D coordinates, occupancy, and B-factor.

FieldMeaning
x, y, z3D coordinates in angstroms
occupancyhow fully that atom is observed in the model
B-factoratomic displacement / uncertainty / flexibility indicator
chainwhich polymer chain the atom belongs to
residue numberposition of residue in that chain
# Python example: read a few ATOM/HETATM records directly. # This shows the raw file format before using a parser. pdb_file = "6u4i.pdb" with open(pdb_file) as handle: shown = 0 for line in handle: if line.startswith(("ATOM", "HETATM")): record = line[0:6].strip() atom_name = line[12:16].strip() resname = line[17:20].strip() chain_id = line[21].strip() resnum = line[22:26].strip() x = float(line[30:38]) y = float(line[38:46]) z = float(line[46:54]) occupancy = float(line[54:60]) bfactor = float(line[60:66]) print(record, atom_name, resname, chain_id, resnum, (x, y, z), occupancy, bfactor) shown += 1 if shown == 5: break
Slide 9

Chains, residues, atoms, occupancy, B-factor

  • Chain: one molecular entity, usually A, B, C...
  • Residue: amino acid or ligand entry
  • Atom: coordinate-bearing item inside a residue
  • Occupancy: less than 1.0 can mean alternate conformations or partial disorder
  • B-factor: higher values often mean more disorder or mobility
# Python example: summarize chains and average B-factor by residue. # This helps you see which parts of a structure are more flexible or uncertain. from Bio.PDB import PDBParser from collections import defaultdict parser = PDBParser(QUIET=True) structure = parser.get_structure("example", "1tim.pdb") model = structure[0] print("Chains in model:", [chain.id for chain in model]) bfactors = defaultdict(list) for atom in structure.get_atoms(): residue = atom.get_parent() chain = residue.get_parent().id reskey = (chain, residue.get_resname(), residue.id[1]) if atom.bfactor is not None: bfactors[reskey].append(atom.bfactor) for reskey, values in list(bfactors.items())[:10]: avg_b = sum(values) / len(values) print(reskey, f"avg B-factor = {avg_b:.2f}")
Slide 10

Crystal structure vs cryo-EM vs predicted model

X-ray crystal

Often high atomic detail, but requires crystals and may include crystal-packing effects. Hydrogens are often not directly seen.

Cryo-EM

Excellent for larger complexes and flexible assemblies. Resolution can vary across the map.

Predicted model

Fast and powerful for fold prediction, but ligand positions, protonation, waters, and catalytic geometry may need extra caution.

For mechanistic studies, an experimental ligand-bound structure is often the best starting point if one exists.

Identify protein chain, ligand, water, ions

# Fully annotated Python example: summarize the contents of a PDB structure. # This is a very practical first script for any beginner. from Bio.PDB import PDBList, PDBParser from collections import Counter pdb_id = "6u4i" filename = f"{pdb_id}.pdb" # Download the file if you do not already have it locally. # Biopython may save with different naming in some versions, so check your folder. pdbl = PDBList() pdbl.retrieve_pdb_file(pdb_id, pdir=".", file_format="pdb", overwrite=True) # If the saved file name differs, rename it manually or replace filename below. # For simplicity, you can also download the PDB file directly from RCSB. parser = PDBParser(QUIET=True) structure = parser.get_structure(pdb_id, filename) model = structure[0] chain_ids = [chain.id for chain in model] print("Chains:", chain_ids) summary = Counter() for residue in structure.get_residues(): hetflag = residue.id[0] if hetflag == " ": summary["protein residues"] += 1 elif hetflag == "W": summary["waters"] += 1 else: summary[f"hetero:{residue.get_resname().strip()}"] += 1 print("\nStructure summary") for key, value in summary.items(): print(f"{key:20s} {value}") # Why this matters: # before studying chemistry, always know what molecules are present.
Slide 13

Locate the active site

Three beginner strategies

  1. Use the bound ligand as an anchor
  2. Look for conserved polar or charged residues around it
  3. Check literature or UniProt annotations after structural inspection

Rule of thumb

The active site is usually the residue cluster within about 4–6 Å of the bound ligand or catalytic metal/cofactor.

# Python example: identify likely active-site residues around ligand atoms. # This version reports only standard amino acids and counts how often # each residue appears near ligand atoms. from Bio.PDB import PDBParser, NeighborSearch from collections import Counter STANDARD_AA = { "ALA","ARG","ASN","ASP","CYS","GLN","GLU","GLY","HIS","ILE", "LEU","LYS","MET","PHE","PRO","SER","THR","TRP","TYR","VAL" } parser = PDBParser(QUIET=True) structure = parser.get_structure("ksi", "6u4i.pdb") ns = NeighborSearch(list(structure.get_atoms())) ligand_atoms = [] for residue in structure.get_residues(): if residue.get_resname().strip() == "EQU": ligand_atoms.extend(list(residue.get_atoms())) counts = Counter() for atom in ligand_atoms: for residue in ns.search(atom.coord, 5.0, level="R"): resname = residue.get_resname().strip() if resname in STANDARD_AA: chain = residue.get_parent().id resnum = residue.id[1] counts[(chain, resname, resnum)] += 1 print("Likely pocket / active-site residues:\n") for key, count in counts.most_common(20): print(key, "contact count", count)

Measure hydrogen-bond distances

Distance is the easiest first measurement. A short donor–acceptor distance can support the presence of a hydrogen bond, but angle and chemistry also matter.

# Python example: measure a specific distance between two named atoms. # Replace the residue numbers and atom names with ones from your structure. from Bio.PDB import PDBParser import numpy as np parser = PDBParser(QUIET=True) structure = parser.get_structure("ksi", "6u4i.pdb") model = structure[0] chain = model["A"] # Example placeholders: # residue 14 TYR atom OH # ligand EQU residue number 201 atom O1 # You must confirm these exact IDs in your file. atom1 = chain[(" ", 14, " ")]["OH"] atom2 = None for residue in chain: if residue.get_resname().strip() == "EQU": if "O1" in residue: atom2 = residue["O1"] break if atom2 is None: raise ValueError("Could not find ligand atom O1; inspect the ligand atom names first.") distance = np.linalg.norm(atom1.coord - atom2.coord) print(f"Distance = {distance:.2f} Å") # Educational note: # This script is intentionally explicit. Beginners should learn to trace exactly # which chain, residue, and atom are being measured.

One useful end-to-end Python mini-workflow

# End-to-end beginner workflow: # 1) parse a structure # 2) list chains # 3) list ligands/waters/ions # 4) find residues near a ligand # 5) report closest atoms to the ligand from Bio.PDB import PDBParser, NeighborSearch import numpy as np pdb_file = "6u4i.pdb" ligand_name = "EQU" parser = PDBParser(QUIET=True) structure = parser.get_structure("example", pdb_file) model = structure[0] print("Chains:", [c.id for c in model]) # List hetero components print("\nHetero components:") for residue in structure.get_residues(): if residue.id[0] != " ": print(residue.get_parent().id, residue.get_resname(), residue.id) # Find ligand atoms ligand_atoms = [] for residue in structure.get_residues(): if residue.get_resname().strip() == ligand_name: ligand_atoms.extend(list(residue.get_atoms())) if not ligand_atoms: raise ValueError(f"Ligand {ligand_name} not found") # Neighbor search for nearby residues ns = NeighborSearch(list(structure.get_atoms())) near_residues = set() for atom in ligand_atoms: for residue in ns.search(atom.coord, 4.0, level="R"): if residue.get_resname().strip() != ligand_name: near_residues.add((residue.get_parent().id, residue.get_resname(), residue.id[1])) print("\nResidues near ligand:") for item in sorted(near_residues): print(item) # Closest 10 protein atoms to the first ligand atom ref = ligand_atoms[0] distances = [] for atom in structure.get_atoms(): if atom.get_parent().get_resname().strip() == ligand_name: continue d = np.linalg.norm(atom.coord - ref.coord) residue = atom.get_parent() distances.append((d, residue.get_parent().id, residue.get_resname(), residue.id[1], atom.name)) print("\nClosest atoms to first ligand atom:") for row in sorted(distances)[:10]: print(row)
